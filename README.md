
# 🧠 Binary Sentiment Analysis App

A Machine Learning-based web application that classifies user input text into **Positive 😊 or Negative 😡 sentiment** using Natural Language Processing (NLP).
“Model trained on real-world review dataset with TF-IDF vectorization and Logistic Regression achieving high accuracy.”

---

## 🚀 Features

* 🔍 Analyze sentiment of text instantly
* ⚡ Fast predictions using trained ML model
* 🌐 Interactive UI built with Streamlit
* 🎯 Binary classification (Positive / Negative)

---

## 🧾 Sentiment Mapping

| Sentiment   | Label |
| ----------- | ----- |
| Negative 😡 | 0     |
| Positive 😊 | 1     |

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/your-username/sentiment-app.git
cd sentiment-app
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the App

```
streamlit run app.py
```
